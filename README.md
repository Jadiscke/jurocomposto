# jurocomposto.net

Website for Brazillian Interest Calculation values based on Future Value Equations.

## Stack

- **Node.js 12.18LTS**
- **Nunjucks**
- **Express.js**
- **HTML, CSS, ES6**

## Getting Started

Clone the repository than use `npm i -y` for installing all the dependecies

## Develop

Use `npm start` for start the developing mode with hot reload using **Nodemon** and **browser-sync**.
**npm-run-all** is used for parallel scripting of **Nodemon** and **browser-sync**.
